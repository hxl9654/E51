# 51单片机模块代码开源计划
目前，基于AVR单片机的Arduino，由于拥有大量器件的驱动函数模块，上手简单而被爱好者追捧。<br/>
本人受此启发，打算做一个51单片机的各类函数模块库，为初学51单片机的爱好者提供便利。<br/>
本项目采用GPL v3许可证授权，任何使用本源代码的项目必须开源，且须注明使用了本项目的代码。<br/>
